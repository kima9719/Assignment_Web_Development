Intranet

This page provides information about Intranet.

  • Intranet: A private network that uses Internet protocol technology.
  • Private Network: A network that is contained within an organization's premises.
  • Internal Communication: The exchange of information within an organization.
  • Secure Access: Controlled access to resources within the organization.
