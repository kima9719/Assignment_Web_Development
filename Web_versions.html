Web Versions

This page lists the different versions of the web application.

  • Web 1.0: Refers to the early stage of the internet where websites where mostly static
  • Web 2.0: Refers to the modern interactive web where users can create and share content
  • Web 3.0: Refers to the decentralized web with blockchain technology and user ownership
  • Web 4.0: Refers to the future vision of the web
