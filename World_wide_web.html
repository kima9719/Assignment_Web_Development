World Wide Web

This page provides information about the World Wide Web.

  • World Wide Web: A system of interconnected documents and resources accessible via the internet.
  • HTML: The standard markup language for creating web pages.
  • HTTP: The protocol used for communication between web browsers and servers.
  • URL: A string of characters used to identify a resource on the internet.
