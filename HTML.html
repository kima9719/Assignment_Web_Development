HTML

This page provides information about HTML (HyperText Markup Language).

  • HTML: The standard markup language for creating web pages.
  • Tag: A keyword enclosed in angle brackets that defines the structure of a web page element.
  • Attribute: A property of an HTML element that provides additional information about the element.
  • Element: A component of an HTML document that defines its structure and content.
